## Перед запуском


```bash

python3 migrate_to_db.py
```

### 4. Проверка статистики

```bash
python3 db_admin.py stats
```

## Запуск бота

### 1. Остановите старую версию

```bash
# Найдите процесс
ps aux | grep bot.py

# Остановите (замените PID)
kill <PID>
```

### 2. Запустите новую версию

```bash
python3 bot.py
```

### 3. Проверьте логи

Должны увидеть:
```
INFO - База данных SQLite инициализирована
INFO - Бот запущен
INFO - Включены уведомления за 10 минут до урока
```

## Функциональное тестирование

### ✅ Тест 1: Новый пользователь

1. Отправьте боту `/start`
2. Выберите корпус
3. Введите группу
4. Проверьте БД: `python3 db_admin.py user <ваш_user_id>`

**Ожидается:** Пользователь добавлен в БД с корпусом и группой.

### ✅ Тест 2: Изменение группы

1. Нажмите "Изменить данные"
2. Выберите другой корпус
3. Введите другую группу
4. Проверьте БД: `python3 db_admin.py user <ваш_user_id>`

**Ожидается:** Данные в БД обновлены, старые данные заменены.

### ✅ Тест 3: Расписание

1. Отправьте `/schedule`
2. Проверьте, что расписание отображается

**Ожидается:** Бот загружает группу из БД и показывает расписание.

### ✅ Тест 4: Уведомления

1. Нажмите кнопку 🔔
2. Уведомления должны выключиться
3. Нажмите еще раз
4. Уведомления должны включиться
5. Проверьте БД: `python3 db_admin.py user <ваш_user_id>`

**Ожидается:** Поле `notifications_enabled` меняется с 0 на 1.

### ✅ Тест 5: Перезапуск бота

1. Остановите бота (Ctrl+C)
2. Запустите снова: `python3 bot.py`
3. Отправьте боту `/schedule`

**Ожидается:** Бот помнит вашу группу и показывает расписание (без /start).


## Возможные проблемы

### ❌ ModuleNotFoundError: No module named 'telegram'

**Решение:**
```bash
pip install -r requirements.txt
```

### ❌ database is locked

**Причина:** Два экземпляра бота или открытый SQLite клиент

**Решение:**
```bash
# Найти и остановить все процессы
ps aux | grep bot.py
kill <PID>
```

### ❌ Пользователь не найден после миграции

**Причина:** В pickle не было группы

**Решение:** Попросите пользователя пройти /start заново

### ❌ Уведомления не приходят

**Проверьте:**
1. `python3 db_admin.py user <user_id>` - уведомления включены?
2. Есть ли группа и корпус?
3. Правильное ли время урока?
4. Работает ли JobQueue?
